Isolation

A branch and a worktree each

Every agent gets its own private checkout, so parallel edits can't collide. An agent is pinned to its worktree after spawn — even a shell profile that jumps directories can't walk it into your real working tree.

Parallel

Up to nine live terminals

Real terminals, tiled automatically. Start a swarm and watch them all at once, drag to reorder, and get a desktop ping when a backgrounded agent finishes or needs you.

Landing

Merge, apply, or discard

Read an agent's diff against your base branch and merge it, take just the files you want onto your branch, or throw the whole branch away — without leaving the app.

Why it's safe

Five agents. One repo. Nothing trampled.

Running several agents on one checkout normally means they overwrite each other's edits — and yours. Monad gives each agent its own branch and its own git worktree: a real, separate directory on disk, backed by the same repository.

Your working tree is never one of them. Agents can't see it, can't reach it, and can't be tricked into it. When one gets the answer right, you review its diff and land it deliberately — nothing reaches your branch until you say so.

Get Monad

Free and MIT-licensed. Runs on your machine against your repo — no account, no telemetry, and no inference cost, because it drives the agent CLIs you already pay for.

01

Install an agent CLI first. Monad orchestrates the tools you already run — it doesn't ship a model of its own. Have at least one of claude, codex, gemini or cursor-agent on your PATH, then open a git repo and add your first agent.

macOS

Apple silicon · .dmg

Download .dmg

Drag Monad to Applications, then run this once. macOS flags unsigned downloads as “damaged”; this clears the quarantine flag.

xattr -dr com.apple.quarantine /Applications/Monad.app

Windows

x64 · installer

Run the installer. If SmartScreen appears, choose More info → Run anyway — the build isn't code-signed yet.
